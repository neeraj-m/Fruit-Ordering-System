Fruit Ordering System

by neeraj-m

Ordering Instructions:

Type one of the following in the console to interact with the ordering system:

newOrder() - Create a new order, which will save and close the previously open order. Note: the first order is created automatically on first loading of the page.

addToOrder(<string>type, <integer>quantity) - to add any number of 'Banana', 'RedApple', 'GreenApple', or 'Melon' to the current order.

removeFromOrder(<string>type, <integer>quantity) - to remove any number of, 'Banana', 'RedApple', 'GreenApple', or 'Melon' from the current order.

printCurrentOrder() - prints the full details of the current order for only the fruits ordered.

Helper Functions:

Type one of the following in the console to get help:

printCurrentOrder() - returns the full current order list.

showFruitTypes() - returns the specific fruit types available to order.

saveAndPrintOrderList() - saves the current order, and then prints the order list for every order added.

Example Console Usage:

addToOrder("RedApple", 7);
addToOrder("Banana", 3);
addToOrder("Melon", 1);
removeFromOrder("RedApple", 1);

newOrder();

addToOrder("GreenApple", 3);
addToOrder("Banana", 1);
removeFromOrder("GreenApple", 1);
addToOrder("RedApple", 1);

saveAndPrintOrderList();
