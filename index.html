Fruit Ordering System

Ordering Instructions:

Type one of the following in the console to interact with the ordering system:

newOrder() - Create a new order, which will save and close the previously open order. Note: the first order is created automatically on load.

addToOrder(<string>type, <integer>quantity) - to add either any number of, 'Banana', 'RedApple', 'GreenApple', or 'Melon' to the current order.

printCurrentOrder() - prints the full details of the current order for only the fruits ordered.

Help Functions:

Type one of the following in the console to get help:

showFruitTypes() - returns the specific fruit types available to order.
